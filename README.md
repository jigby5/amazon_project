# To run the project:

# Backend:
##### Enter backend/Amazon.api/Amazon.api
##### Open the command line
##### Run "dotnet run"
##### API will be operational on port 5000 (HTTPS) or 4000 (HTTP)

# Frontend:
##### Enter frontend/Amazon\ Simulation
##### Install dependencies with "npm install"
##### Run frontend with "npm run dev"
##### Open localhost:3000 for access to the site
